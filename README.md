### **Title:** Summary of Wisconsin's Yearly Wildfire Season

### **Author:** *Nicholas Kositzke*

### **Dependencies:** 
[Leaflet 1.9] (https://leafletjs.com)
[Chart.js 4.4.1] (https://cdn.jsdelivr.net/npm/chart.js)
[Wiscondin DNR Wildfire Data (2025)] (https://dnrmaps.wi.gov/wildfiredashboard/)


### **Code Version:** Last updated: March 11, 2026

### **Description:** This interactive proportional symbol map, built using Leaflet, demonstrates the shifting pattern of wildfire concentrations across an 11-month timespan in Wisconsin during the year 2025.
